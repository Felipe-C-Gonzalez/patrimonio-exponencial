# Patrimonio Exponencial

## 🎨 Identidade Visual

Decisões de design focadas em legibilidade e profissionalismo.

- **Background (Light):** `#F1F5F9`
- **Primary (Ação):** `#0A9E5A`
- **Surface (Card):** `#FFFFFF`
- **Typography:** [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts)

## 📱 Responsividade & Acessibilidade
- Layout totalmente fluido adaptado para Mobile, Tablet e Desktop.
- Uso de atributos ARIA para tecnologias assistivas.
- Validação nativa de formulários.
